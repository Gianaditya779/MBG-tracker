Cara Integrasi Google Sheets

1
Buat Google Spreadsheet

Buka Google Sheets, buat spreadsheet baru dengan kolom: Tanggal, Nama Guru, Kelas, Jumlah Diterima, Jumlah Dibagikan, Sisa, Status, Waktu Input

2
Buat Google Apps Script

Di spreadsheet, buka Extensions → Apps Script, paste kode berikut:

function doPost(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet()
              .getActiveSheet();
  var data = JSON.parse(e.postData.contents);
  sheet.appendRow([
    data.tanggal, data.namaGuru, data.kelas,
    data.diterima, data.dibagikan, data.sisa,
    data.status, data.waktuInput
  ]);
  return ContentService
    .createTextOutput(JSON.stringify({success:true}))
    .setMimeType(ContentService.MimeType.JSON);
}
3
Deploy sebagai Web App

Klik Deploy → New deployment → Web App. Set "Who has access" ke "Anyone". Copy URL yang dihasilkan.

4
Tempel URL ke script.js

Di file script.js, ganti nilai variabel APPS_SCRIPT_URL dengan URL deployment Anda.
